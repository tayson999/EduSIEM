Lightweight SIEM / Log Analysis is a mini Security Information and Event Management (SIEM) system designed for learning and hands-on practice.

The project can:

Parse Apache, SSH, and Windows logs using regular expressions

Aggregate events into time windows (time-series)

Detect anomalies using statistical methods (z-score) and rule-based detection

Generate structured alerts in JSON and CSV formats

Be easily extended and configured thanks to its modular design
